# Fougere

Petit outil graphique en Python/Pygame pour choisir une couleur RGB.

## Description

Ce projet fournit une interface avec des sliders et des champs numériques pour modifier les composantes rouge, verte et bleue d'une couleur.

Fonctionnalités principales :

- sélection d'une couleur en RGB ;
- sliders interactifs ;
- champs numériques pour les valeurs R, G et B ;
- presets de couleurs ;
- affichage de la couleur choisie ;
- conversion vers HEX et HSL.

## Installation

Créer un environnement virtuel, puis installer les dépendances :

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Sous Linux/macOS :

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Lancement

Lance le fichier Python principal du projet, par exemple :

```bash
python main.py
```

Si le fichier principal porte un autre nom, adapte la commande. Oui, GitHub ne devine pas encore les intentions humaines, scandale technologique.
